# Retail Sales Analysis Dashboard (Excel)

This project demonstrates a simple business dashboard built using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## Dashboard Features
- KPI showing Total Sales
- Sales by Product chart
- Sales by Region chart
- Interactive Region filter

## Key Insights
- Laptop generated the highest revenue.
- South region produced the highest sales.
- Chair had the lowest sales.
